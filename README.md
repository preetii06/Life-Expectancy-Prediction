# 📊 Life Expectancy Prediction using Machine Learning

## 📌 Overview
This project predicts life expectancy using machine learning by analyzing health, economic, and social factors. The dataset is sourced from WHO and spans multiple countries from 2000 to 2015.

---

## 🎯 Objective
- Identify key factors affecting life expectancy  
- Build predictive models  
- Compare performance of different models  

---

## 📂 Dataset
- Source: WHO Global Health Observatory  
- Time Period: 2000–2015  
- Features include GDP, mortality, BMI, schooling, immunization, etc.

---

## ⚙️ Technologies Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Statsmodels  

---

## 🔍 Workflow
1. Data loading using Pandas  
2. Data preprocessing (missing values, encoding, scaling)  
3. Exploratory Data Analysis (EDA)  
4. Model building (Regression, Random Forest, Gradient Boosting)  
5. Model evaluation using R², MAE, MSE, RMSE  

---

## 📈 Results
- Achieved R² score of ~0.97–0.99  
- Random Forest and Gradient Boosting performed best  

---

## 💡 Key Insights
- Higher mortality reduces life expectancy  
- Higher GDP and education increase life expectancy  
- Immunization improves overall health outcomes
